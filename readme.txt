== Description ==

eBayFlashstore is a plugin allowing to display eBay auctions from a unique eBay store, in a flash carrousel interface for any Wordpress page.
Just select some parameters to match your needs and your listing will be automatically displayed in the module. This module uses the the latest API available on Ebay developer website.
Our plugins are eBay compatible application certified.

1. For eBay store only. Push all sales available from your store in all contries at the same time, or filter by country.
2. flash external interface: all request and flash files are located on our server, so updates and improvements are automatically available on your own widgets.
3. Multiple versions of the widget can be done on the same pages.
3. cachable in the parameter module
4. Efficient and designed social sharing for facebook. Let your users publish your auction on facebook, or do it yourself on your own wall.

5. External interface: widget will always be available from our server
6. All parsing and code running on our server: your server stays free of surcharge. Only rendering view is executed on Wordpress website.
7. Widget call by javascript: validated code by W3C
8. CSS editable
9. five languages.
10. Also available for other CMS.


== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Define your own parameters in the configuration page of the plugin

== Parameters ==

1. EBAY STORE USERNAME : Your store name on eBay, used to sell your items. (not your eBay username. If you don't have a store, use the eBayflahseller widget)
2. TYPES OF ITEM :  AllItemTypes, AuctionItemsOnly, FixedPricedItem
3. SORT ORDER : Current price, time, bid count.
4. NUMBER OF AUCTIONS : Number between 1 and 100
5. LANGUAGES : 6 languages available.
6. LOCALIZATION : localize your auctions by country in the 11 ebay different websites
